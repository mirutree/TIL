### const & let 소개
 - 새로운 변수 선언 방식
 - 블록 단위 {}로 변수 범위 제한
 - const : 한번 선언한 값에 대해 변경할 수 없음 (상수 개념)
 - let : 한번 선언한 값에 대해 다시 선언할 수 없음

###  [ES5의 주요 특징] 변수 스코프와 호이스팅
 - ES5는 {}에 산관없이 스코프가 설정됨
 ```
 var sum = 0;
 for (var i=1; i <= 5; i++) {
  sum = sum + i;
 }
console.log(sum); //15
console.log(i);   // 6
```
